Welcome to Nigel Seecharan's Simple Api

Log in in to create, or delete newly created employees or tasks. Please note that default entries do not have the option to be modified.

This api is for demo purposes, and has routes that access small lists of data related to employees and tasks.

Route structure goes as follows: [website url]/api/[emoloyees or tasks]

For more information on the routes, please refer to the project readme.
